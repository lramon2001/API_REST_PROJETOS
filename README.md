 ## Para testar a aplicacão
 
1-Baixe a release do projeto [aqui]();
 
2-Altere as propriedades do banco de dados no application.properties;
 
3-Entre no diretório que tenha o arquivo pom.xml;
 
4-Execute o comando:
```bash
 
 mvn spring-boot:run
 
```
